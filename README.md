Luxury black & gold website built with plain HTML, CSS, and JavaScript.
Portfolio loads dynamically from Supabase (table: `projects`), with placeholder fallback.

## Files
- `index.html` — main entry
- `styles.css` — design system & layout
- `script.js` — interactions, Supabase fetch, particles, testimonials
- `assets/logo.png` — brand logo
- `vercel.json` — deployment config
